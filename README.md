# 🚁 AutoDrag – Drone Club

## 📌 Project Overview

**AutoDrag – Drone Club** is a multi-page static website built using HTML and CSS.
It represents a realistic college club platform used for **showcasing achievements, events, and student recruitment**.

This project was developed as part of a **college assignment** to demonstrate real-world website structure and navigation.

---

## 🌐 Features

* 🏠 **Home Page** – Introduction and tagline (“Ready to Fly!”)
* 📖 **About Page** – Club information, achievements, and purpose
* 🎯 **Events Page** – Highlights of competitions and participation
* 📝 **Join Page** – Registration form for new members
* 🔗 Navigation bar for seamless page switching

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Internal Styling)

---
